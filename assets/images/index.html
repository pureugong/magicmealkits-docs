    Documentation Assets¶

    This directory contains assets for the Docker-based documentation.

    Logo and Favicon¶

    To complete the documentation setup, you need to add the following files:

    1. logo.png - The logo displayed in the navigation bar (recommended size: 128x128px)
    2. favicon.png - The favicon displayed in browser tabs (recommended size: 32x32px)

    You can add your own logo files by placing them in this directory with the names specified above.

    Example Logo Creation¶

    If you don't have a logo yet, you can create a simple one using tools like:

    • Canva - Free online design tool
    • GIMP - Free image editor
    • Inkscape - Free vector graphics editor

    Placeholder Logo¶

    Until you add your own logo, the documentation will use its default icon. To quickly create a simple logo, you can use online tools like:

    • LogoMakr
    • FreeLogoDesign
    • Hatchful by Shopify

    Remember to create both a logo.png and favicon.png file for the best documentation appearance.
